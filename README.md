## 👋 About Me

I'm a first-year **Information Technology** student in Oman, building production-grade software at the intersection of **full-stack web development**, **Arabic-first RTL interfaces**, and **workflow automation**.

I design and ship complete systems end to end — from database schema to UI to deployment — using modern TypeScript tooling, and I have a particular focus on document-automation pipelines that turn raw data into polished, branded Arabic PDF/Word output at scale. I'm increasingly working with AI-assisted engineering workflows, using coding agents to design, build, and ship multi-step systems autonomously.

- 🌍 Fluent in **English**, **Arabic**, and **Urdu**
- 🎓 IELTS Academic Band **7.5 (C1)**
- 🎯 Interests: AI Engineering · Software Engineering · Data Science · Computer Vision · Automation
- 🚀 Goal: internships and a long-term career in the US / European tech industry
- 💬 Ask me about: React/TypeScript architecture, Arabic RTL UI, tRPC, or PDF/DOCX generation pipelines
